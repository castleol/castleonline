缺省按键（支持游戏手柄）
跳跃：Z,B1
攻击：X,B2
防守：C,B3
后退：V,B4
左移：左键,LEFT
右移：右键,RIGHT
蹲下：下键,DOWN
对话：上键,UP
投掷：攻击+上
跳下：跳跃+下
